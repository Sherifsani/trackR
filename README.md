# trackR

Workforce productivity and payroll platform for remote teams. Employers track employee work sessions, browser activity, and productivity patterns — then disburse salaries directly to bank accounts based on verified hours worked.

## What it does

**For admins (employers/HR)**
- Invite employees via email and set hourly rates, break allowances, and overtime multipliers
- View live activity feeds as employees work — domains visited, time per category, idle periods
- Review session analyses generated by Gemini AI after each clock-out: productivity score, grade, highlights, and concerns
- Access cross-session pattern insights: peak hours, fatigue profile, day-of-week trends, and behavioral anomalies
- Disburse salaries to employee bank accounts via Squad — calculates regular vs overtime pay per session and initiates bank transfers

**For employees**
- Clock in and out from the web dashboard or the browser extension
- View their own session history, productivity scores, and AI-generated insights
- Set up verified bank details for payroll (account lookup via Squad)

## Browser Extension

The [trackR Monitor extension](https://github.com/Sherifsani/trackR-extension) handles activity tracking. It runs as a Chrome MV3 service worker that:

- Tracks active tab domain, title, and dwell time
- Categorises sites automatically (development, design, comms, meetings, docs, off-task, etc.)
- Records idle and focus-lost periods
- Syncs activity to the server every minute, with an eager 5-second flush after each tab visit
- Discovers clock-in/out state by polling the API — works even when the web app is served through a tunnel

## Tech stack

- **Framework**: Next.js 15 App Router, TypeScript
- **Database**: PostgreSQL via Prisma ORM
- **Styling**: Tailwind CSS v4, shadcn/ui, HugeIcons
- **AI**: Google Gemini 2.5 Flash — per-session analysis and cross-session pattern insights
- **Payments**: Squad (Nigerian fintech) — bank account verification, wallet top-up, salary disbursement

## Getting started

```bash
npm install
cp .env.example .env   # fill in your credentials
npx prisma migrate dev
npm run dev
```

### Environment variables

```
DATABASE_URL=
JWT_SECRET=
GEMINI_API_KEY=
SQUAD_BASE_URL=https://sandbox-api-d.squadco.com
SQUAD_SECRET_KEY=
SQUAD_MERCHANT_ID=
SITE_URL=http://localhost:3000
```

### Demo data

Seed a ready-to-use admin + employee with 3 weeks of sessions, payments, and AI insights:

```bash
curl -X POST http://localhost:3000/api/dev/demo
```

Seed two additional employees with a full week of activity for payroll disbursement demos:

```bash
curl -X POST http://localhost:3000/api/dev/demo-users
```
